English:
These files do not belong to the project under development, they are needed for the functioning of the marketplace. These files contain the authorization, registration, account and messenger source code.

For the project to work, place the marketplace files in the "app" folder, then import the "unesell.sql" database located in this folder, and change the configuration file to "/service/config.php".

Enter the path to the project files on your local or web hosting in the 3rd level domain. For example "web/app/store/ [project files]". The project has been launched.


Русский:
Эти файлы не относятся к разрабатываемому проекту, они нужны при функционировании маркетплейса. Эти файлы содержат исходный код авторизации, регистрации, аккаунта и мессенджера.

Для работы проекта поместите файлы меркетплейса в папку "app", затем импортируйте базу данных "unesell.sql", которая находится в этой папке, и измените конфигурационный файл в "/service/config.php".

Пропишите в локальном или веб-хостинге в домене 3 уровня путь к файлам проекта. Например, "web/app/store/ [файлы проекта]". Проект запущен.
